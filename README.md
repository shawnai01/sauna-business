# 사우나 사업 리서치 🧖

사우나/목욕 사업의 본격적인 사업성 검토를 위한 리서치 프로젝트.

## 목적
- 사우나 사업의 시장 구조, 수익 모델, 초기 투자비, 운영 경제학을 체계적으로 분석
- 창업 의사결정에 필요한 근거 자료 생성

## 팀 구성
| 에이전트 | 역할 | 지침 파일 |
|----------|------|-----------|
| **Project Lead** | 리서치 방향 설정, 요청 관리 | `project-lead.md` |
| **Research Director** | 수합, 편집, 퀄리티 관리 | `research-director.md` |
| Market Analyst | 시장 환경, 소비 트렌드, 규제 | `market-analyst.md` |
| Industry Analyst | 사우나 산업 구조, 경쟁, 벤치마크 | `industry-analyst.md` |
| Business Modeler | 수익구조, 투자비, P&L, 시나리오 | `business-modeler.md` |

## 데이터
| 파일 | 용도 |
|------|------|
| `project-note.md` | Shawn의 사업 관심사·방향성·제약조건 |
| `notes/YYYY-MM-DD.md` | 상세 리서치 노트 아카이브 |

## 산출물
- **요약 브리핑**: 텔레그램 전달. 핵심 인사이트 중심.
- **상세 리서치 노트**: `notes/YYYY-MM-DD.md`에 저장.

## 실행 방식
- **트리거**: Shawn이 리서치 주제/질문을 던지면 → 조사 실행
- 크론잡 없음. Shawn의 요청이 트리거.
- 리서치 범위: 한국 중심 (글로벌 벤치마크 참조)
